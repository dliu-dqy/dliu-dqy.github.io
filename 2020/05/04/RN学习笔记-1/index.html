构成1 React

react是一个开源的JS框架，用于构建用户界面。

react的依赖库

核心库：react.min.js

DOM相关库：react-dom.min.js
什么是DOM？：文档对象模型，一种标准编程接口，将文档视作一个树形结构，树的每个结点表示了一个HTML标签或标签内的文本项。

react的最小可分单位是元素

新添加一个div标签，标签交由根DOM节点管理，通过函数

如何刷新元素

元素是不可变的，更改的方法是创建一个新的元素，并传入render方法。
